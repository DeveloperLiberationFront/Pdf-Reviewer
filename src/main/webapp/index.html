Pdf Reviewer

How to use PdfReviewer for reviewers:

  1. Login with GitHub
  2. Download the paper
  3. Add comments to the paper
    • Add tags within curly braces ({})
  4. Upload the file and submit the review

How to use PdfReviewer for writers:

  1. Login with GitHub
  2. Select the repository containing your paper
  3. Select your paper in the repository
  4. Select reviewers
  5. Submit the review request
